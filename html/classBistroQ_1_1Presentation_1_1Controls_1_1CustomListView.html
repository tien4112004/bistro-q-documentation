Detailed Description

A custom ListView implementation that supports multiple selection with observable collection binding and messaging capabilities. Inherits from ListView and implements messaging and disposal interfaces.

Template Parameters
TThe type of items contained in the ListView.

Enhances the base ListView by:

  • Supporting two-way binding for selected items
  • Implementing messaging for selection changes
  • Managing proper cleanup of resources
