Detailed Description

View model for the admin account creation page. Handles the logic for adding new admin accounts and managing related data such as zones and tables.

This class implements INavigationAware for navigation handling and IDisposable for cleanup. It uses the MVVM pattern with ObservableRecipient as its base class.
