Detailed Description

ViewModel for managing the cashier's table view interface. Coordinates between zone overview, table grid, and order details components. Handles real-time checkout notifications and state management.

Implements multiple interfaces for functionality:

  • ObservableObject for MVVM pattern
  • INavigationAware for page navigation
  • IRecipient for handling checkout messages
  • IDisposable for resource cleanup
