Detailed Description

ViewModel responsible for managing the order cart functionality in the client interface. Handles order creation, modification, and status tracking with support for nutrition facts.

Implements:

  • ObservableRecipient for MVVM pattern
  • IRecipient for handling cart and order messages
  • IDisposable for resource cleanup
