Detailed Description

ViewModel responsible for managing a time counter with elapsed time display. Provides functionality for starting, stopping, and resetting a timer, as well as formatting the elapsed time for display.

Implements ObservableObject to provide change notifications for timer properties. Uses DispatcherTimer for UI thread-safe time updates.
