Detailed Description

ViewModel responsible for managing the order item grid display and interactions. Handles data loading, pagination, status filtering, and message-based updates.

Implements:

  • ObservableObject for MVVM pattern
  • IDisposable for resource cleanup
  • IRecipient interfaces for handling various message types
