Detailed Description

A JSON-based storage service for authentication tokens, implementing the ITokenStorageService interface. This service stores access and refresh tokens in a JSON format, providing methods to save, retrieve, and clear tokens. This approach may be useful for scenarios where tokens need to be persisted in a lightweight, human-readable format.
