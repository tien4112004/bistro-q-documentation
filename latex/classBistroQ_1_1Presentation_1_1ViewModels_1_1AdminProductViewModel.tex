\doxysubsection{Detailed Description}
View\+Model for managing products in the admin interface. Handles listing, adding, editing, and deleting products with support for pagination and sorting. 

Implements multiple interfaces for functionality\+:
\begin{DoxyItemize}
\item Observable\+Recipient for MVVM pattern
\item INavigation\+Aware for page navigation
\item IDisposable for resource cleanup
\item IRecipient for handling pagination messages 
\end{DoxyItemize}
