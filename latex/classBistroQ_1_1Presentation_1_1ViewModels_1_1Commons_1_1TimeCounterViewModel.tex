\doxysubsection{Detailed Description}
View\+Model responsible for managing a time counter with elapsed time display. Provides functionality for starting, stopping, and resetting a timer, as well as formatting the elapsed time for display. 

Implements Observable\+Object to provide change notifications for timer properties. Uses Dispatcher\+Timer for UI thread-\/safe time updates.
